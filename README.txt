# Python Quiz App

This is a simple Python quiz application that allows users to take a quiz, receive feedback on correct and incorrect answers, and track high scores. The quiz questions are presented in random order, and the user's score is calculated at the end.

## Features

- User-friendly interface
- Customizable quiz questions
- Input validation for user responses
- Feedback on correct and incorrect answers
- High score tracking and display
- Average score calculation
- percentage score calculation

## Prerequisites

- Python 3.x

## How to Use

1. Download the Zip file to your local machine:
2. Navigate to the project directory:
3. Run the quiz app:(main.py)

NOTES:

4. Follow the on-screen instructions to take the quiz:
- Enter your name.
- Specify the number of questions you want to answer.
- Answer each question by entering the corresponding number (1 to 4) for your choice.
- Receive feedback on correct and incorrect answers, along with the correct answers.
- At the end of the quiz, see your score and percentage.
- Decide if you want to take the quiz again.



## Customization

You can customize the quiz questions by editing the `quiz_app.py` script. Look for the section where questions are defined and add or modify questions as needed.

## High Scores

The app tracks high scores for different users. After each quiz, your username and score will be added to the high scores list. The top 5 high scores are displayed at the end of the program.
